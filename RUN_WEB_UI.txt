💡 TROUBLESHOOTING:

  Port already in use?
    → Kill other process: lsof -ti:5001 | xargs kill -9
    → Or use custom port: PORT=8080 python3 web_server.py

  Can't connect?
    → Check server is running in terminal
    → Try http://127.0.0.1:5001

  Need help?
    → Read QUICK_DEMO.md for full guide
